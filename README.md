# Jumpyvi Morebrew

## How do I install these formulae?

`brew install jumpyvi/morebrew/<formula>`

Or `brew tap jumpyvi/morebrew` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jumpyvi/morebrew"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
